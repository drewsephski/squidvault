# Frontend Design Workflow

This workflow uses the `frontend-design` skill to create visually striking, memorable web interfaces that avoid generic "AI slop" aesthetics.

## When to Use

Use this workflow when:
- Building new UI components, pages, or applications
- Redesigning existing interfaces
- Creating landing pages, dashboards, or marketing sites
- Needing a bold, distinctive aesthetic direction

## Usage

Simply invoke the skill and describe what you want to build:

```
@frontend-design
Create a landing page for a creative agency with a brutalist aesthetic
```

## What You'll Get

The skill will guide you through creating:
- **Bold aesthetic direction** - Commit to a clear concept (minimalist, maximalist, brutalist, retro-futuristic, etc.)
- **Distinctive typography** - Beautiful, unique font pairings with fluid sizing
- **Cohesive color palette** - Modern CSS color functions, tinted neutrals, sharp accents
- **Asymmetric layouts** - Visual rhythm through varied spacing, intentional grid breaks
- **Purposeful motion** - Well-orchestrated page loads with staggered reveals
- **Production-ready code** - Working React/Next.js components with exceptional attention to detail

## Design Principles Applied

- **Typography**: Modular type scales, distinctive font pairings
- **Color**: OKLCH color functions, perceptually uniform palettes
- **Layout**: Asymmetric compositions, container queries, fluid spacing
- **Motion**: Exponential easing, transform/opacity animations
- **Interaction**: Progressive disclosure, optimistic UI, intentional interactions

## Example Prompts

- "Create a dark mode SaaS dashboard with retro-futuristic vibes"
- "Build a minimalist portfolio site for a photographer"
- "Design a maximalist e-commerce product page"
- "Make a brutalist blog layout with stark typography"

Remember: The skill will interpret creatively and make unexpected choices. Commit fully to a distinctive vision.
